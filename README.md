# 🎓 Student Management System (MySQL Project)

## 📌 Project Overview

This project is a database system to manage:

* Students
* Courses
* Enrollment
* Exams
* Results

## 🛠️ Technologies Used

* MySQL
* SQL (DDL, DML, Joins, Aggregations)

## 🗂️ Database Tables

* Students
* Courses
* Enrollment
* Exams
* Results

## 🔥 Features

* Track student enrollments
* Manage exams and results
* Calculate average marks
* Identify top-performing students

## 📊 Key Queries

* List all students
* Students enrolled in a course
* Highest scoring student
* Students not enrolled
* Average marks per course

## 🚀 How to Run

1. Open MySQL
2. Run `project.sql`
3. Execute queries
